Multiple Linked Lists

Implement a singly linked list having all unique elements with the following operations.

  • II 00 xx Inserts element xx at the end.
  • II 11 yy xx If the element yy exists, then insert element xx after the element yy, else insert element yy before the existing element xx. Assuming either the element xx or the element yy exists.
  • II 22 zz yy xx Inserts element xx in the middle of the elements zz and yy. The element zz appears before the element yy.
  • UU xx pp Links the next pointer of the element xx to the node lying at the pthpth position from the element xx while traversing towards right. In case of insufficient number of nodes, the counting continues by updating the existing linked list to its circular version.

Input:

  • Line 1 contains an integer NN indicating the total number of operations.
  • Each of the following NN lines contains an operation to be performed in the format as is described above.

Output:

  • Line 1 has 11 if the linked list gets updated to its circular version, else 00.
  • Line 2 has a count of the number of nodes whose addresses are contained in the next pointer of multiple nodes. If output at Line 2 is zero then output Line 4 will not be printed.
  • Line 3 has space separated contents of all the nodes which are counted to get the output at Line 2 in increasing order. If output at Line 2 is zero then output Line 3 will have space separated contents of the generated linked list.
  • Line 4 has space separated respective frequencies of each output value, say xx, at Line 3. 'Frequency of each output value xx' means the count of multiple nodes whose next pointers have address of a node with this value xx.

Constraints

  • All integers range in between 1 and 1000.

Sample Input I:

3
I 0 1
I 1 1 7
I 2 1 7 3

Sample Output I:

0
0
1 3 7
